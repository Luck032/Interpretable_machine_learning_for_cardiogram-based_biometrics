# Interpretable Machine Learning for Cardiogram-Based Biometrics

This repository contains Python code used to reproduce the results presented in the paper "[Interpretable machine learning for cardiogram-based biometrics](https://doi.org/10.48550/arXiv.2510.19775)" authored by Ilija Tanasković (ORCiD: [0000-0002-6488-4074](https://orcid.org/0000-0002-6488-4074)), Ljiljana B. Lazarević (ORCiD: [0000-0003-1629-3699](https://orcid.org/0000-0003-1629-3699)), Goran Knežević (ORCiD: [0000-0001-8951-3774](https://orcid.org/0000-0001-8951-3774)), Nikola Milosavljević (ORCiD: [0000-0001-5061-149X](https://orcid.org/0000-0001-5061-149X)), Olga Dubljević (ORCiD: [0000-0003-1560-1661](https://orcid.org/0000-0003-1560-1661)), Bojana Bjegojević (ORCiD: [0000-0002-8421-5572](https://orcid.org/0000-0002-8421-5572)) and Nadica Miljković (ORCiD: [0000-0002-3933-6076](https://orcid.org/0000-0002-3933-6076)).

## GitHub Repository Contents
This repository contains Python and R programming codes, as well as extracted timestamps for segments that describe emotional states and feature sets for both ECG and ICG recordings that reproduce results for the preprint "[Interpretable machine learning for cardiogram-based biometrics](https://doi.org/10.48550/arXiv.2510.19775)".
Also, this repository contains [README.md](https://github.com/Luck032/Interpretable_machine_learning_for_cardiogram-based_biometrics/blob/main/README.md) file with relevant information essential for code reproducibility and [LICENSE](https://github.com/Luck032/Interpretable_machine_learning_for_cardiogram-based_biometrics/blob/main/LICENSE) file that contains license information that covers shared software codes.
